Excel Inventory & Sales Management System

## Project Overview
 
 This project is an Excel-based Inventory and Sales Management System developed to track stock movement, monitor daily transactions, and maintain accurate inventory records for garment business operations.
 
 The system is designed to automate stock calculations, reduce manual errors, and support monthly reconciliation processes through structured data linking and automated formulas.

 ## Project Purpose
 
 The purpose of this project is to create a reliable and efficient system that helps businesses:
 
 Track daily inventory movement
 
 Monitor stock availability
 
 Record production and returns
 
 Maintain operational accuracy
 
 Support reconciliation and reporting
 
 This system reflects real-world garment inventory workflow used in business operations.
